## âš¡ Quick Start // Initialization

### Prerequisites
* Node.js 18+
* npm or yarn
* An OpenAI API Key (for ORBIT functionality)

### 1. Clone the Repository

```bash
git clone [https://github.com/Mayank8159/ITFarmer.git](https://github.com/Mayank8159/ITFarmer.git)
cd ITFarmer/frontend



### 2. Install Dependencies

```bash
npm install
# or
yarn install

```

### 3. Configure Environment

Create a `.env.local` file in the root of the `frontend` directory:

```env
OPENAI_API_KEY=sk-your-openai-api-key-here

```

### 4. Ignite the Server

```bash
npm run dev

```

Access the system at `http://localhost:3000`.
